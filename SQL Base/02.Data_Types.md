# Что такое SQL?

## Определение

SQL (Structured Query Language) — язык структурированных запросов.

Он используется для общения с базой данных.

Мы не программируем базу данных.

Мы отправляем ей команды (запросы).

Например:

SELECT * FROM Users;

означает

"Покажи мне всех пользователей."

# Типы SQL-команд

SQL делится на четыре большие группы.

## 1. DDL (Data Definition Language)

DDL работает со структурой базы данных.

То есть изменяет саму базу.

Команды:

CREATE

ALTER

DROP

TRUNCATE

Запомнить:

DDL = структура.

---

## 2. DML (Data Manipulation Language)

Работает уже с данными.

Команды:

SELECT

INSERT

UPDATE

DELETE

Запомнить:

DML = данные.

---

## 3. DCL (Data Control Language)

Управление правами доступа.

Команды:

GRANT

REVOKE

---

## 4. TCL (Transaction Control Language)

Работа с транзакциями.

Команды:

COMMIT

ROLLBACK

SAVEPOINT

---

# Чем отличается DDL от DML?

DDL

Изменяет структуру базы данных.

Например:

CREATE TABLE

DROP TABLE

ALTER TABLE

DML

Работает уже с данными внутри таблицы.

Например:

SELECT

INSERT

UPDATE

DELETE

🔥 Это один из самых популярных вопросов на Junior-собеседованиях.

---

# Типы данных

При создании таблицы необходимо указать, какой тип данных будет храниться в каждом столбце.

Например:

Возраст — число.

Имя — текст.

Дата рождения — дата.

Цена — число с копейками.

---

# Самые важные типы данных

INT

Целые числа.

Используется для:

id

age

quantity

---

VARCHAR

Текст переменной длины.

Используется для:

имени

фамилии

email

названия товара

---

CHAR

Текст фиксированной длины.

Например:

UA

NL

US

---

DECIMAL

Дробные числа с высокой точностью.

Используется для:

денег

цен

баланса

Не использовать FLOAT для денег.

---

DATE

Хранит дату.

Например:

2026-07-19

---

TIMESTAMP

Дата и время.

Например:

2026-07-19 18:25:15

Часто используется для:

created_at

updated_at

---

BOOLEAN

Хранит только:

TRUE

FALSE

---

# CREATE DATABASE

Пример

CREATE DATABASE Store;

---

# CREATE TABLE

Создает новую таблицу.

Общий шаблон

CREATE TABLE table_name (

    column_name DATA_TYPE,

    column_name DATA_TYPE

);

---

Пример

CREATE TABLE Products (

    id INT,

    name VARCHAR(100),

    price DECIMAL(10,2),

    quantity INT

);

---

После выполнения создается пустая таблица.

Данные в ней пока отсутствуют.

---

# Типичные ошибки

❌ Писать FLOAT для денег.

Правильно:

DECIMAL(10,2)

---

❌ Ставить запятую после последнего столбца.

Неправильно

age INT,

)

Правильно

age INT

)

---

❌ Хранить возраст как VARCHAR.

Возраст — INT.

---

❌ Хранить цену как VARCHAR.

Цена — DECIMAL.
